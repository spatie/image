# Supported formats

The following image formats are tested and supported by the package. If your GD/Imagick installation supports a format that is not listed you are probably fine to use it but your mileage may vary.

| Format | GD | Imagick |
|--------|----| --- |
| jpeg   | ✅  | ✅ |
| png    | ✅  | ✅ |
| gif    | ✅  | ✅ |
| webp   | ✅  | ✅ |
| heic   | ❌  | ✅ |
| tiff   | ❌  | ✅ |
